Atributo de Classes e identificação

Ums dos atriburos mais usados é o atributos ID e o Class. Sendo o class="" atributo de classe e o id="" identificador.
Em um documento HTML, o identificador nao pode se repetir. Pôs, ele é usado para identificar uma tag.
Ja a classe, pode ser usada diverssas vezes, montando assim um grupo.
E uma tag pode ter mais de uma classe.
Ex:

fundo vermelho

fundo vermelho

fundo azul

Esses identificadores são importante, para a introdução do CSS e do Java.






Tag Style e CSS


Tag stayle

A tag de meta dado Style (se é meta dado, logo, se encontra no head do HTML) comporta codigo CSS. E como o proprio nome ja diz, codigo esse, focado na estilização do documento HTML.


CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo usada para definir a apresentação visual de documentos HTML. Em outras palavras, enquanto o HTML estrutura o conteúdo de uma página web, o CSS é responsável por estilizar esse conteúdo, determinando como ele será exibido aos usuários.

Com o CSS, você pode alterar a cor do texto, o fundo, o espaçamento entre parágrafos, criar layouts complexos, ajustar imagens e muito mais. Ele permite separar o conteúdo da apresentação visual, tornando o código mais limpo e fácil de manter.
EX:

fundo amarelo
Letra marrom
